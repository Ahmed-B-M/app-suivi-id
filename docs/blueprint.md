# **App Name**: Urbantz Data Exporter

## Core Features:

- API Data Retrieval: Fetch data from the Urbantz API using the provided API key, date range, and hub IDs.
- Data Filtering: Filter tasks based on specified parameters (date range, hub).
- JSON Export: Save the retrieved and filtered task data into a JSON file for local storage.
- Automated Scheduling: A tool using generative AI to determine the best schedule for API calls and JSON export, considering potential rate limits and server load.
- Error Logging: Implement comprehensive error logging to track API request failures and data processing issues.

## Style Guidelines:

- Primary color: Deep blue (#30475E) to convey reliability and data focus.
- Background color: Light gray (#F0F0F0) to provide a clean, neutral backdrop.
- Accent color: Teal (#26A69A) for interactive elements and key indicators.
- Body and headline font: 'Inter' sans-serif font for clear data presentation. This font is used both in headlines and for the body text.
- Code font: 'Source Code Pro' for configurations such as API keys and other app details
- Use a tabular layout for presenting any summary data or logs.
- Simple, outline-style icons for visual cues in data processing steps.
